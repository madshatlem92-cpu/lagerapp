# 📦 Varetelling App

En enkel og mobilvennlig varetellingsapp laget med HTML og JavaScript.

Appen kjører direkte i nettleseren og fungerer perfekt på mobil med kamera for strekkodeskanning.

---

## ✅ Funksjoner

- 📷 Scan strekkoder med mobilkamera
- 📦 Registrer varer med antall
- 🔍 Søk etter varer
- ➕ Legg til +1 raskt
- ➖ Trekk fra lager
- ❌ Slett varer
- 📊 Eksporter lager til Excel (CSV)
- 💾 Automatisk lagring (localStorage)
- 📱 Mobilvennlig (kan legges til som app)

---

## 📱 Hvordan bruke appen

1. Åpne appen i nettleser
2. Trykk **📷 Start scanning**
3. Scan strekkode
4. Legg inn navn (første gang)
5. Skriv antall
6. Trykk **Registrer**

👉 Neste gang fylles varen automatisk inn

---

## 📊 Excel eksport

- Trykk **📊 Last ned Excel**
- Du får en `.csv` fil
- Kan åpnes direkte i Excel

---

## 💾 Lagring

Alle data lagres lokalt i nettleseren:

- Ingen database nødvendig
- Ingen login
- Data blir liggende på enheten din

---

## ⚠️ Viktige ting

- Lageret er lokalt på enheten (ikke delt)
- Bruker du en annen telefon → tomt lager
- Sletting av nettleserdata vil også slette lageret
